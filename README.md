Nome do projeto

Sistema de Automação de Relatórios com Python

Descrição

Sistema desenvolvido em Python para leitura de arquivos CSV e Excel, tratamento de dados, cálculo de indicadores, geração automática de gráficos e exportação de relatórios em HTML e Excel.

Tecnologias
Python
Pandas
Matplotlib
OpenPyXL
Funcionalidades
leitura de planilhas
limpeza de dados
cálculo de KPIs
gráficos automáticos
exportação em Excel
relatório em HTML
